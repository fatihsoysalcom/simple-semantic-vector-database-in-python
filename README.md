# Simple Semantic Vector Database in Python

This example demonstrates a lightweight, zero-dependency vector database built from scratch in Python. It showcases how text is converted into mathematical coordinates (embeddings) and how semantic search is performed using cosine similarity, highlighting the difference between traditional keyword matching and vector-based semantic search.

## Language

`python`

## How to Run

Run the script using Python: `python main.py`
